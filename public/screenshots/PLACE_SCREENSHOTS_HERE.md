# Screenshot Guide — MarketSentinel UI

Place all screenshots in this folder.
Each page can have **multiple screenshots** (scroll states, modals, etc.)

## Naming Convention

Use descriptive suffixes after the page name:
- `dashboard-1.png` → overview state
- `dashboard-2.png` → top opportunities detail
- `market-signals-1.png` → grid view
- `market-signals-2.png` → list view / filter active
- etc.
